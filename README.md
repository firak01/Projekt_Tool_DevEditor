# Projekt_Tool_DevEditor
Tools for development assistant

Module:
- Eclipse start: Setze Umgebungsvariablen (u.a. für andere Module wichtig).

- Git :             Tools zum Ein-/Auschecken von Änderugen in Projekten 
                     Auch Tools zum Kopieren von Projekten in einen Zielordner und umgekehrt.

- RexEx Replace:     Ersetze Text (hier Code) in Java Dateien.

- TextDatei Sucher: Suche in mehreren Dateien und per RegEx definierte Zeilen werden in der Konsole ausgegeben.
